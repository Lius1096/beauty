# Beauty Shop Backend + Frontend Starter

## Structure
- `src/`: backend in TypeScript using Express + Mongoose.
- `public/`: simple HTML front-end for booking.
- `.env.example`: example environment variables.

## Quick start
1. Copy `.env.example` to `.env` and set MONGO_URI & JWT_SECRET.
2. `npm install`
3. `npm run dev`
4. Visit `http://localhost:3000/public/index.html`
